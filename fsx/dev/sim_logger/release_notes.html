Sim_logger release notes

  • 2.31 BUGFIX for non-English FSX for unverified IGC file saves to
    <user documents>\<local language FSX Files folder>\sim_logger_unverified_logs
    from the hard-coded 'Flight Simulator X Files' used in 2.30 (ok in English, broken elsewhere).
  • 2.30 Changed the default folder for unverified IGC file saves to
    (My) Documents\Flight Simulator X Files\sim_logger_unverified_logs
    from the FSXBASE sub-folder used in 2.29 which clashed with Vista/Win7 security.
  • 2.29 Fix for Plan-G flightplan files - these can now be read successfully so the task can be included in the tracklog (there was a unicode incompatibility with prior versions of sim_logger).
    To provide better support for logging in 'free flight', these tracklogs will be saved in FSXBASE\Modules\sim_logger\free_flight_logs. With previous versions of sim_logger, these tracklogs may have been discarded.
  • 2.27 Modified 'Restart tracklog from here' menu option so that the 'flight start time' used in the tracklog filename is updated to the current time that option was selected. This means a subsequent tracklog save on the same flight will not overwrite earlier saved tracklogs.
  • 2.26 Added an 'enable/disable autosave' option to the sim_connect.ini file, so that the automatic save of IGC tracklogs can be disabled (e.g. after an aircraft crash, FSX crash-to-desktop, or the end of a flight longer than 20 minutes that has not been saved)
  • 2.25 First production version 2, with improved recording of FSX soaring-related checksums, and a 'replay' function for previously saved IGC tracklog files.
  • 1.18 Final production version 1, saving FSX aircraft tracklog to an IGC-format file including some FSX checksums.
