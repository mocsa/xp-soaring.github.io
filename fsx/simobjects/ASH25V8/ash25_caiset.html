=============== Installation for FSX ================

English

Copy the folder ASH25V8_CAISET include the subfolders MODEL , PANEL , SOUND and TEXTURE under your \FSX\SIMOBJECTS\AIRPLANES folder.

There is a file named 'gps-nav.dat', a file with waypoints for the gps unit. This file must be in the main FSX folder, otherwise FSX shuts down while you switch on the GPS unit.

Copy this file to this folder, only if there is no 'gps-nav.dat' file.

For the function of the Caiset instruments the following is to be done:

  1. Install FSUIPC4 for FSX.
    Download FSUIPC from http://www.schiratti.com/dowson.html.
    The installation produces the file FSUIPC.dll in the FSX\Modules folder
  2. The following files must be placed in the main FSX folder:
    FSSound.dll and SOARRec.dll
    They can be downloaded from my site: http://www.fsglider.de/fsdll1.zip
  3. VISTA and Windows 7 users pay attention: to have the caiset running errorfree, FSX.exe has to be opened with administrator-rights !
