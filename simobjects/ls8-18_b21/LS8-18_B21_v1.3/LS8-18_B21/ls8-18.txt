    =============================== Installation for FSX ==================================================== 

 
    English
    -------
    Copy the folder LS8-18 include the subfolders MODEL , PANEL , SOUND and TEXTURE
    under your \FSX\SIMOBJECTS\AIRPLANES folder.
 

 
    There is a file named 'gps-nav.dat', a file with waypoints for the gps unit. This
    file must be in the main FSX folder, otherwise FSX shuts down while you
    switch on the GPS unit.
    Copy this file to this folder, only if there is no  'gps-nav.dat' file.
  
    For the function of the Caiset instruments the following is to be done: 

    1.) Install FSUIPC4 for FSX.
        to receive on 
        http://www.schiratti.com/dowson.html 
        the installation produces the file FSUIPC.dll in the FSX\Modules folder

    2.) In the main FSX folder the following files must be:
        FSSound.dll and SOARRec.dll 
        to receive on my site at
        http://www.fsglider.de/fsdll1.zip
 
        VISTA-users pay attention: to have the caiset running errorfree, 
        FSX.exe has to be opened with administrator-rights !

         zu erhalten unter

   2.)  Im Hauptordner des FSX müssen sich folgende Dateien befinden:
        FSSound.dll und SOARRec.dll 
        zu erhalten auf meiner Seite unter

   Français
   --------
 
   Copier le dossier LS8-18 (y compris les sous-dossiers Model, Panel, Sound et Texture)
   dans le répertoire \FSX\SIMOBJECTS\AIRPLANES 

  
   Il y a un dossier appelé 'gps-nav.dat '. Ce dossier doit être dans le répertoire principale de FSX,
   autrement FSX s'arrête tandis que vous alimentez l'unité de GPS. Copier ce dossier à ce répertoire,
   seulement s'il n'y a aucun dossier de'gps-nav.dat '.

   Pour la fonction, ce qui suit est nécessaire :

   1.) Installer FSUIPC4 pour FSX.
       Ce programme peut téléchargé à : 
       http://www.schiratti.com/dowson.html 
       l'installation produit le fichier FSUIPC.dll dans le répertoire FSX\Modules

   2.) Vérifier que les fichiers suivants sont présents dans le répertoire principale du FSX :
       FSSound.dll et SOARRec.dll 
       Ces fichiers peuvent téléchargés de ma site
       http://www.fsglider.de/fsdll1.zip

       Attention VISTA-utilisateurs : FSX.exe doit être ouvert avec des droits d'administrateur !

     Notes:

     Use SHIFT + E for open/close the canopy
     Use SHIFT + E + 2 for open/close the canopy window

     SHIFT + E öffnet/schließt die Haube
     SHIFT + E + 2 öffnet/schließt das Haubenfenster

     Utilisez MAJ + E pour ouvir/fermer la verrière
     Utilisez MAJ + E + 2 pour ouvir/fermer la fenêtre de la verrière
