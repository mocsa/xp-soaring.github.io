Flight Logging

Objective

An easy-to-use and secure logger is available that records the pilots cross-country flight, including the waypoints of the set task, such that the tracklog can be analyzed to show successful completion of the flight and the associated completion time.

If the IGC format is available as a log format, then real-world gliding analysis software can be used for the tracklog analysis.

Technology

X-Plane provides a logging capability (details to follow).

sim_logger provides this capability for FSX.
